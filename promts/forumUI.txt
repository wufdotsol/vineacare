    ## Data Classes
    Class Post:
        postId: string => unique id for the post
        authorId: string => uid of the author
        content: string => content of the post
        timestamp: DateTime => time of the post
        likes: List<String> => list of uids of the users who liked the post
        comments: List<String> => list of uids of the users who commented on the post
        shares: List<String> => list of uids of the users who shared the post
        bookmarks: List<String> => list of uids of the users who bookmarked the post

    Class Comment:
        commentId: string => unique id for the comment
        postId: string => uid of the post
        authorId: string => uid of the author
        content: string => content of the comment
        timestamp: DateTime => time of the comment
        likes: List<String> => list of uids of the users who liked the comment

    Class Chat:
        chatId: string => unique id for the chat
        user1Id: string => uid of the first user
        user2Id: string => uid of the second user
        messages: List<Message> => list of messages in the chat

    Class Message:
        messageId: string => unique id for the message
        chatId: string => uid of the chat
        senderId: string => uid of the sender
        content: string => content of the message
        timestamp: DateTime => time of the message
        likes: List<String> => list of uids of the users who liked the message

## Actions Definitions
- Liking a post: adds the current user's uid to the likes list of the post in the firebase firestore database in the posts collection and adds the post's id to the myLikes list of the current user in the firebase firestore database in the users collection.
- Commenting on a post: adds the comment id to the comments list of the post in the firebase firestore database in the posts collection and adds the comment to the firebase firestore database in the comments collection.
- Bookmarking a post: adds the current user's uid to the bookmarks list of the post in the firebase firestore database in the posts collection and adds the post's id to the myBookmarks list of the current user in the firebase firestore database in the users collection.
- Sending a friend request: adds the current user's uid to the sentFriendRequests list of the other user in the firebase firestore database in the users collection and adds the other user's uid to the recievedFriendRequests list of the current user in the firebase firestore database in the users collection.
- Accepting a friend request: adds the other user's uid to the myFriends list of the current user in the firebase firestore database in the users collection and adds the current user's uid to the myFriends list of the other user in the firebase firestore database in the users collection.
- Rejecting a friend request: removes the other user's uid from the recievedFriendRequests list of the current user in the firebase firestore database in the users collection and removes the current user's uid from the sentFriendRequests list of the other user in the firebase firestore database in the users collection.
- Removing a friend: removes the other user's uid from the myFriends list of the current user in the firebase firestore database in the users collection and removes the current user's uid from the myFriends list of the other user in the firebase firestore database in the users collection.
- Deleting a post: removes the post's id from the myPosts list of the author in the firebase firestore database in the users collection and deletes the post from the firebase firestore database in the posts collection.
- Sending a message: adds the message to the messages list of the chat in the firebase firestore database in the chats collection and adds the chat's id to the myChats list of the sender in the firebase firestore database in the users collection.
- Sending a message instanciates a chat object with a unique id, the two users' uids, and an empty list of messages.    
- the unique chat id is added to the myChats list of both users in the firebase firestore database in the users collection.
- the message is added to the messages list of the chat in the firebase firestore database in the chats collection.
- a chat is only created when a message is sent for the first time between two users.
- chats display the other user's pfp, display name, username, and the last message sent in the chat.
